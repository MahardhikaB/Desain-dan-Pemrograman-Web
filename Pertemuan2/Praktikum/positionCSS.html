psition: static;

Position static digunakan untuk mengatur element menjadi statis secara default. Elemen akan mengikuti posisi normal sacara default, elemen tidak dipengaruhi oleh properti seperti top, bottom, left dan right.:

Elemen ini diatur menggunakan posisi: static;




position: relative;

Sebuah elemen HTML yang menggunaan position relative akan terletak pada posisi normal. Mengatur properti seprti atas, kanan, bawah, dan kiri dari elemen yang diposisikan "relatif" akan membuatnya jauh dari posisi normalnya. Konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut.:

Emen ini diatur dengan menggunakan posisi : relative;



position: fixed;

Sebuah element HTML yang di setting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman wesbite di scroll. Berlaku pengaturan left, bottom, top, dan right pada position fixed:

Elemen ini diatur denga menggunakan posisi : fixed;















position: absolute

Element HTML yang menggunakan posisi absolute akan diposisikan relatif dengan element lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal.:

Elemen ini diatur dengan posisi : relative;
Elemen ini diatur dengan posisi : absolute;
